# Exobase Private Limited - Website

A modern, responsive website for Exobase Private Limited, an India-based export house operating in the agri and natural commodities sector.

## Features

- **Home Page**: Hero section, key highlights, products preview, certificates showcase, and CTA
- **About Us Page**: Company story, approach, values, and management information
- **Products Page**: Detailed information about rice, cotton, maize, wheat, and sugar
- **Contact Us Page**: Contact form and company information
- **Certificates Section**: Highlighted certifications and compliance documents
- **Responsive Design**: Mobile-friendly across all devices
- **Modern UI/UX**: Beautiful design using brand colors (#3493BF and #004B7D)

## Technology Stack

- React 18.2.0
- React Router DOM 6.20.0
- CSS3 with custom properties
- Modern responsive design

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Build for Production

```bash
npm run build
```

This creates an optimized production build in the `build` folder.
